编写简单的 IRC 机器人

首先简单介绍下 IRC。IRC，中文名可译作 “互联网中继聊天”，是 1988 年 8 月由芬兰人 Jarkko Oikarinen 创造的用以取代 MUT 的网络聊天协议。IRC 主要用于在不同的频道（Channel）中进行群聊，但同样也可以用于个人对个人的私聊。不同平台下的 IRC 客户端可以参考中文维基。

再介绍下套接字（Socket）：套接字是为特定网络协议（如 TCP/IP、ICMP/IP、UDP/IP 等）套件对上的网络应用程序提供者提供当前可移植标准的对象。它们允许程序接受并进行连接，如发送和接受数据。为了建立通信通道，网络通信的每个端点都需要建立一个套接字对象，这一点极为重要。

套接字为 BSD UNIX 系统核心的一部分，而且他们也被许多其他类 UNIX 操作系统包括 Linux 所采纳。许多非 BSD UNIX 的系统（如Windows，Mac OS X 等）都以库形式提供了对套接字的支持。三种最流行的套接字类型是:stream、datagram 以及 raw。stream 和 datagram 套接字可以直接与 TCP 协议进行接口，而 raw 套接字则接口到 IP 协议。但套接字并不限于 TCP/IP。

而 IRC 是一种公开的协议，采用了 TCP 和 SSL 协议。IRC 也是一个分布式的客户端/服务器结构（C/S结构）。通过IRC 客户端连接到一个 IRC 服务器，就可以访问这个服务器以及它所连接的其他服务器上的频道。 要使用 IRC，必须先登陆到一个 IRC 服务器上，最常用的 IRC 服务器为 irc.freenode.net。

也就是说，我们制作 IRC 机器人的第一步，是通过 python 与 IRC 服务器进行通信。为了实现此目的，需要使用 python 中的 socket 模块（套接字模块），利用该模块包含的函数和类定义，就可以实现符合 IRC 协议的网络通信。

通常来讲，使用 python 建立一个简单的 IRC 客户端连接需要三个步骤：

  1. 创建 socket 对象，调用 socket 构造函数：

    socket = socket.socket(family, type)

    family的值可以是 AF_UNIX（Unix域，用于同一台机器上的进程间通讯），也可以是 AF_INET（对于IPV4协议的TCP和 UDP），至于 type 参数，则可以使用 SOCK_STREAM（流套接字）、 SOCK_DGRAM（数据报文套接字）或者 SOCK_RAW（raw套接字）。

  2. 使用 socket 的 connect 方法连接服务器：

    socket.connect((host,port))

    host 和 port 分别对应所要连接的 IRC 主机名及端口号。

  3. 保持客户端和服务器之间的通信

通常来说 IRC 客户端还需要进行关闭的第四步，不过我们的目的是尽可能的保持机器人在线，需要关闭时通过手动 Ctrl + C 关闭即可。

编写机器人所需的基础知识就是这些，下面我们通过具体的代码来实现机器人。首先设定环境编码并导入 socket 模块：

#!/usr/bin/env python
#-*- encoding: utf-8 -*-
 
import sys
reload(sys).setdefaultencoding("utf8")
 
import socket

接着设置一些常用变量：

irc_host = "irc.freenode.net"
irc_port = 6666
bot_name = "bzzzt"
irc_chan = "#linuxba"

以上变量分别对应所要连接到的 IRC 主机和端口、机器人的名字，以及所要加入的频道，可以根据自己的实际情况分别进行修改。接着就可以开始创建 irc_sock 对象了：

irc_sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)

然后使用 irc_sock 的 connect 方法连接至 IRC 服务器：

irc_sock.connect((irc_host, irc_port))

连接成功后向服务器发送机器人的身份信息并加入聊天室：

irc_sock.send("NICK " + bot_name + "\r\n")
irc_sock.send("USER " + bot_name + " " + bot_name + " " + bot_name + " :" + bot_name + "\r\n")
irc_sock.send("JOIN " + irc_chan + "\r\n")

这里说明一下，IRC 用户需要分别向服务器提供昵称（Nick）、用户姓名（Username）以及真实姓名（Realname），而我们这里的机器人全部设定为相同的名字，如果想要分别设定可以参照下列格式进行修改（添加到常用变量）：

irc_sock.send("NICK " + Nick + "\r\n")
irc_sock.send("USER " + Username + " " + Username + " " + Username + " :" + Realname + "\r\n")

接着保持客户端与服务器的连接，用 irc_sock 的 makefile() 方法从服务器接收信息、逐行分析，并从终端打印出来，同时响应服务器发送的 PING 口令：

while True:
    data = irc_sock.makefile()
 
    for line in data:
        print line
        if line.startswith("PING"):
            send_back = line.replace("PING", "PONG")
            irc_sock.send(send_back)

这里再说明一下，当 IRC 客户端连接到服务器后在 4、5 分钟左右没有向服务器发送任何信息时，服务器会向客户端发送格式为 PING :SERVER 的口令，客户端必须对此做出回应，否则服务器会判定客户端已掉线而关闭连接，回应格式为 PONG :SERVER 的响应信息即可。

至此，我们就完成了一个简单的 IRC 机器人的编写，完整代码可以参考这里（复制粘贴或另存为 simple_ircbot.py 后即可使用，运行环境 Python 2.7）。若想了解详细的 IRC 协议，请查阅 rfc1459。
